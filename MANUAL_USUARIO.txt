FLUJO DE USO RECOMENDADO
=========================

PASO 1: REGISTRAR ESTUDIANTES
   1. Accede al sistema con docente/docente123
   2. Menu → Estudiantes - MAESTRO
   3. Ingresa nombres (APELLIDO, Nombre)
   4. Completa datos (cédula, teléfono)
   5. Los datos se copian automáticamente

PASO 2: INGRESAR CALIFICACIONES
   1. Menu → Ingresar Calificaciones
   2. Selecciona: Grado → Materia → Trimestre → Componente
   3. Ingresa nota (1.0 a 5.0)
   4. El promedio se calcula automáticamente

PASO 3: CONTROL DE ASISTENCIA
   1. Menu → Control de Asistencia
   2. Selecciona grado y fecha
   3. Marca P (Presente), A (Ausente), T (Tardanza)
   4. El sistema cuenta ausencias

PASO 4: GENERAR REPORTES
   1. Menu → Resúmenes / Dashboard / Reportes
   2. Visualiza promedios y estadísticas
   3. Genera alertas de bajo rendimiento

PASO 5: IMPRIMIR LIBRO
   1. Menu → [🖨️] IMPRIMIR LIBRO DE REGISTRO
   2. Documento se prepara para impresión
   3. Incluye todas las notas del año
   4. Con carátula oficial MEDUCA
   5. Entregar a supervisión


VALIDACIONES AUTOMÁTICAS
=========================

✓ NOTAS: 1.0 a 5.0 (decimal con punto)
✓ CÉDULA: 8 dígitos formato panameño
✓ TELÉFONO: Formato válido (XXXX-XXXX)
✓ FECHAS: Formato válido (DD/MM/YYYY)
✓ NOMBRES: No pueden estar vacíos

RESPALDOS
=========

RESPALDO AUTOMÁTICO:

1. Menu → Configuración
2. Click "Crear Respaldo"
3. Se guarda como: Respaldo_YYYYMMDD_HHMM.xlsm

RECOMENDACIÓN: Respaldo semanal + copia trimestral


SOLUCIÓN DE PROBLEMAS
====================

❌ Nota no se guarda → Verifica esté entre 1.0 y 5.0
❌ Contraseña olvidada → Contacta administrador
❌ Datos no aparecen → Presiona Ctrl+S y recarga
❌ Nota: "Hoja no encontrada" → Cierra y reabre el archivo
