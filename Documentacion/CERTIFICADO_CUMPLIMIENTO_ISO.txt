INSTRUCCIONES PARA CAMBIAR CONTRASEÑAS
═══════════════════════════════════════

IMPORTANTE: Cambiar contraseñas de demostración ANTES de usar en producción

Pasos:
1. Abrir RegistroDoc_PRO.xlsm
2. Presionar ALT + F11 (Abre editor VBA)
3. Expandir "Módulos"
4. Seleccionar "RD_SEGURIDAD"
5. Buscar función "IniciarSesion()"
6. Reemplazar:
   - "admin123" con nueva contraseña
   - "docente123" con nueva contraseña
   - "director123" con nueva contraseña
7. Guardar (CTRL + S)
8. Cerrar editor VBA
9. Guardar Excel (CTRL + S)

RECOMENDACIONES:
  ✓ Usar contraseñas >8 caracteres
  ✓ Mezclar mayúsculas, minúsculas, números
  ✓ No usar información personal (cédula, teléfono)
  ✓ Cambiar cada 90 días (admin), 180 días (otros)
  ✓ Guardar en lugar seguro (contraseña principal)
