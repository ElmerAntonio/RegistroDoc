══════════════════════════════════════════════════════════════════
PASO 2 — ACTIVAR LOS BOTONES (MACROS VBA) — Solo una vez
══════════════════════════════════════════════════════════════════
Para que los botones del Menú funcionen debes hacer esto UNA SOLA VEZ:

  a) Con el archivo abierto, presiona:  Alt + F11
     (Se abre el Editor de Visual Basic)

  b) En el menú del editor, haz clic en:  Insertar → Módulo
     (Aparece una ventana en blanco)

  c) Abre el archivo ModuloRegistro.bas con el Bloc de Notas:
     → Clic derecho sobre el archivo .bas
     → "Abrir con" → "Bloc de notas"

  d) Selecciona todo el texto:  Ctrl + A
     Cópialo:                   Ctrl + C

  e) Vuelve al editor VBA (barra de tareas)
     Pega el código en la ventana en blanco:  Ctrl + V

  f) Cierra el editor VBA:  Alt + Q

  g) Guarda el archivo como .xlsm (con macros):
     → Archivo → Guardar como
     → Tipo: "Libro de Excel habilitado para macros (*.xlsm)"
     → Nombre: RegistroDoc_Multigrado_v1
     → Guardar

  h) Cierra y vuelve a abrir el archivo .xlsm guardado
     → Clic en "Habilitar contenido" si aparece barra amarilla

¡Listo! Todos los botones del Menú Principal funcionan ahora.

EMPEZAR A USAR EL SISTEMA
══════════════════════════════════════════════════════════════════
  1. Ve a la hoja CONFIGURACIÓN (pestaña amarilla)
     → Cambia el nombre del docente, escuela y año

  2. Ve a la hoja MAESTRO (pestaña azul oscuro)
     → Escribe los nombres de los estudiantes
     → Columna B = 7°,  Columna D = 8°,  Columna F = 9°

  3. Ve al PANEL_NOTAS para ingresar calificaciones
  4. Ve al PANEL_ASISTENCIA para registrar asistencia diaria
  5. Los RESÚMENES y REPORTES se actualizan automáticamente
