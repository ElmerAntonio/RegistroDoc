PASO A PASO RÁPIDO
==================

1. ABRIR SISTEMA
   → Doble click en RegistroDoc_PRO.xlsm

2. LOGIN
   Usuario: docente
   Contraseña: docente123
   (O admin/admin123 para acceso total)

3. INGRESA ESTUDIANTES
   → Menu: Estudiantes - MAESTRO
   → Escribe nombres en celdas amarillas
   → Los datos se copian automáticamente

4. INGRESA CALIFICACIONES
   → Menu: Ingresar Calificaciones
   → Selecciona grado, materia, trimestre
   → Ingresa notas (1.0 a 5.0)

5. GENERA LIBRO DE REGISTRO
   → Menu: [🖨️] IMPRIMIR LIBRO DE REGISTRO
   → Sistema prepara documento oficial
   → Imprime y entrega a supervisión
